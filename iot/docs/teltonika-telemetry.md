# How to parse Teltonika telemetry data (Codec 8E) with .NET?

Teltonika has implemented many formats and ways to serialize data, in a condensed and optimized way called [Codecs](https://wiki.teltonika-gps.com/view/Teltonika_AVL_Protocols).
There are different parsers or .NET SDK implementations.

You will find here a personal implementation to cover the uncovered aspects by the exiting implementations.
