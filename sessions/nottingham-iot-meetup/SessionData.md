# Title

IoT Device Simulation Tools for Development

# Description

At some point, every IoT solution needs to test device provisioning, integration and interactions, telemetry, as well as other data flows. It is also important to understand how the solution behaves as a whole in response to specific messages, events, or situations.

Testing all of this with real devices is not always easy in terms of logistics and organization (e.g., firmware or the devices themselves being developed in parallel), and it can be costly. That is why using software device simulators is a very flexible approach that saves time, money, and headaches.

This session starts by reviewing the main IoT principles (no need to be an expert to attend) before presenting one of the simulators I have developed and have been using. Other tools will also be mentioned as part of this approach, as I truly enjoy finding ways to help developers make their lives a bit easier.
