# AI Tools for IoT

IoT Simulators and Azure IoT Hub REST API façade have been completed with different types of AI tools, in order to take advantage of the capabilities the new AI offers.

# Skills

The `Skiils` created can be used with:
 - the IoT Simulator MCP Server
 - the Azure IoT Hub MCP Server

 They are also referenced in `agents` and packaged in a `GitHub Copilot plugin`.
